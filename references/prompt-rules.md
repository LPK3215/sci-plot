# 科研绘图提示词生成核心规则

从 paper-craft-skills 提取并适配为通用跨平台提示词规范。

---

## 图片规格

- 竖版 2:3 比例（方法图解默认）
- 横版 16:9 或 4:3（幻灯片/演示场景）
- 所有标注文字清晰可读
- 高清出图

---

## 构图原则

### 信息层次
1. **标题区**：页面顶部，简短说明本页主题（如"方法总览""核心机制：注意力计算"）
2. **主体区**：视觉图解占页面 75%-85% 面积
3. **标注区**：关键节点旁的文字解释，简洁精确

### 视觉流向
- 明确的方向性（左→右，或上→下，或中心向外扩散）
- 用箭头、线条、数字序号引导阅读顺序
- 不允许出现"不知道该看哪"的混乱布局

### 重点强调
- 核心创新点用**强调色**标记（边框加粗/填充色/高亮区域）
- 非关键辅助信息用浅色处理，不抢焦点

### 信息密度
- 机制图不能只画大标题和几个孤立模块；必须包含足够的内部步骤、局部放大或小例子
- 画面空白超过 25% 时，优先补充：关键公式短标注、输入/输出维度、矩阵小格、token 示例、before/after 对比、注意力热力图
- 一页无法容纳则拆成多页，不牺牲可读性
- 封面图允许更概念化；细节图必须优先解释"怎么做"

---

## 文字标注规范

### 标注内容
- 每个方框/节点旁标注名称（如"Embedding""Self-Attention""FFN"）
- 关键数据维度（如"d=512""n=12 heads"）
- 重要的数学关系用简短文字（如"Q×K^T""÷√dk"），不用完整 LaTeX 公式
- 中文页：标题 ≤16 个汉字，标签 ≤8 个汉字

### 标注位置
- 紧贴所标注的对象
- 不遮挡关键视觉元素
- 多个标注不重叠

### 标注语言
- 专业术语保留英文（如"Transformer""Attention""Embedding"）
- 解释性文字用中文
- 标注文字精简——能用 3 个字不用 10 个字

---

## 颜色使用

### 限制原则
- 主色 + 强调色，不超过 3 种颜色
- 不用渐变色、不用过饱和色
- 颜色用于区分，不用于装饰

### 颜色含义
- 深色/粗线 = 核心路径/关键结构
- 浅色/细线 = 辅助信息/上下文
- 强调色 = 本页要让人记住的那一个点

### 各风格强调色参考
| 风格 | 强调色 |
|------|-------|
| paper-figure | 蓝（#2563EB）/ 橙（#D97706）选 1-2 种 |
| sketchnote | 深蓝（#2C5F8A）/ 珊瑚红（#D85C4A）/ 橄榄绿（#5C6B4F）/ 柔和黄（#F3D36B） |
| journal-minimal | Nature 蓝 / IEEE 蓝 / 冷灰 |
| warm-notes | 软蓝 / 珊瑚 / 橄榄 / 黄色 |
| business-research | 炭灰色 + 藏蓝/墨绿/金色单强调 |

## 每页必含

- [ ] 页面顶部标题
- [ ] 清晰且占画面主体的核心视觉元素（流程图/结构图/对比图）
- [ ] 关键节点的文字标注
- [ ] 方向性引导（箭头/数字/线条）
- [ ] 本页唯一的核心强调点（用颜色/大小突出）
- [ ] 机制细节页：至少包含一个局部放大、小例子、维度说明或关键公式短标注

---

## 每页禁止

- [ ] 不要出现任何代码块
- [ ] 不要照片级写实渲染
- [ ] 不要 3D 立体效果
- [ ] 文字标注不要连成一段话（每个标注独立、简短）
- [ ] 不要画"装饰性"元素（和内容无关的图案、花纹）
- [ ] 不要大面积空白
- [ ] 不要蓝底白线工程图纸风
- [ ] 不要复制原论文图的布局（要重新组织信息）
- [ ] 不要做成 PPT 封面或营销海报
- [ ] 不要出现假 logo、假 UI、乱码标签、假品牌名
- [ ] 不要赛博朋克霓虹、廉价 3D、塑料质感
- [ ] sketchnote 不要画成旧羊皮纸、泛黄档案或复古日记

## 提示词结构化格式

每页提示词用以下格式生成，保存在 `prompts/NN-{slug}.md`。

**文件结构说明**：中文说明、构图描述、内容元素是给人看的解释，**English Prompt 是核心产出，直接复制到生图工具即可出图**。

```markdown
# 第 N 页：{页面标题}

## 中文说明
- **类型**：{流程分解图 / 架构总览 / 机制细节 / 对比图 / 数据图}
- **风格**：{paper-figure / sketchnote / journal-minimal / warm-notes / business-research}
- **语言**：{zh / en / bilingual}
- **比例**：{2:3 / 16:9 / 4:3}

## 构图描述
{用自然语言描述画面布局：左/右/上/下/中的空间安排，箭头方向，模块位置关系}

## 内容元素
{列出要画的具体模块、关系、标注}

---

## ▶ English Prompt（核心，直接复制）
{完整的英文提示词，可直接复制到生图工具}

---

## 复制到生图工具
将上方的 "English Prompt" 部分完整复制到以下工具的输入框：
- **ChatGPT（DALL-E）**：直接粘贴
- **豆包**：文生图入口
- **Qwen（通义万相）**：文生图入口
- **Midjourney**：在 `/imagine` 后粘贴
```
