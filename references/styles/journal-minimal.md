# journal-minimal — Nature/IEEE 极简学术风

目标：简约但不简陋，像 Nature / IEEE 论文图和正式学术汇报的结合。内部讨论、答辩或公开汇报都适用。

## 氛围
- 投稿级清晰度
- 可信、直接、不为装饰而装饰
- 像一个认真的学者做的图

## 视觉规范

### 画布
- 白底或极浅冷灰底（#FFFFFF / #F5F6F8）
- 需要时用细线分隔区域，不用色块

### 图元
- 精确的矢量模块和连线
- 管线图、矩阵块、公式标注、小数据缩略图
- 线条统一粗细，箭头精确

### 文字
- 干净无衬线字体
- 紧凑标签，Figure caption 式的克制
- 标题 + 3-8 个短标签

### 色彩
- 黑 + 冷灰为主
- 一个清楚的学术强调色：Nature 蓝 / IEEE 蓝 / 深青 / 暗红
- 强调色只用在一两处关键位置

## 禁止
- 不要卡通、厚重纹理、假 3D、光泽 SaaS 插图
- 不要装饰性贴纸、巨大空洞的英雄区留白
- 不要超过主色+1 强调色的色彩方案
