# journal-minimal — Nature/IEEE Minimal Academic Style / Nature/IEEE 极简学术风

Goal: minimal but not bare, like the combination of a Nature / IEEE paper figure and a formal academic presentation. Suitable for internal discussion, defense, or public presentations. / 目标：简约但不简陋，像 Nature / IEEE 论文图和正式学术汇报的结合。内部讨论、答辩或公开汇报都适用。

## Atmosphere / 氛围
- Submission-grade clarity / 投稿级清晰度
- Credible, direct, no decoration for decoration's sake / 可信、直接、不为装饰而装饰
- Like a figure made by a serious scholar / 像一个认真的学者做的图

## Visual Specification / 视觉规范

### Canvas / 画布
- White or very light cool gray background (#FFFFFF / #F5F6F8) / 白底或极浅冷灰底（#FFFFFF / #F5F6F8）
- When needed, separate regions with thin lines, not color blocks / 需要时用细线分隔区域，不用色块

### Graphic Elements / 图元
- Precise vector modules and connecting lines / 精确的矢量模块和连线
- Pipeline diagrams, matrix blocks, formula annotations, small data thumbnails / 管线图、矩阵块、公式标注、小数据缩略图
- Uniform line weights, precise arrows / 线条统一粗细，箭头精确

### Text / 文字
- Clean sans-serif font / 干净无衬线字体
- Compact labels, figure caption-like restraint / 紧凑标签，Figure caption 式的克制
- Title + 3-8 short labels / 标题 + 3-8 个短标签

### Color / 色彩
- Black + cool gray as primary / 黑 + 冷灰为主
- One clear academic accent color: Nature blue / IEEE blue / deep teal / dark red / 一个清楚的学术强调色：Nature 蓝 / IEEE 蓝 / 深青 / 暗红
- Accent color used only in one or two key locations / 强调色只用在一两处关键位置

## Forbidden / 禁止
- No cartoon, heavy texture, fake 3D, glossy SaaS illustrations / 不要卡通、厚重纹理、假 3D、光泽 SaaS 插图
- No decorative stickers, giant empty hero-area whitespace / 不要装饰性贴纸、巨大空洞的英雄区留白
- No color scheme exceeding primary + 1 accent / 不要超过主色+1 强调色的色彩方案

## English Prompt Snippet / 英文提示词片段
```
Nature / IEEE-inspired minimal academic presentation diagram, publication-quality scientific figure aesthetic, clean white or light gray background, precise method diagram with fine vector lines, clear restrained scientific color palette (black, gray, one accent: nature blue or teal), compact readable sans-serif labels, credible paper submission visual style, no decoration, no 3D, no cartoon
```
