# journal-minimal — Nature/IEEE Minimal Academic Style

Goal: minimal but not bare, like the combination of a Nature / IEEE paper figure and a formal academic presentation. Suitable for internal discussion, defense, or public presentations.

## Atmosphere
- Submission-grade clarity
- Credible, direct, no decoration for decoration's sake
- Like a figure made by a serious scholar

## Visual Specification

### Canvas
- White or very light cool gray background (#FFFFFF / #F5F6F8)
- When needed, separate regions with thin lines, not color blocks

### Graphic Elements
- Precise vector modules and connecting lines
- Pipeline diagrams, matrix blocks, formula annotations, small data thumbnails
- Uniform line weights, precise arrows

### Text
- Clean sans-serif font
- Compact labels, figure caption-like restraint
- Title + 3-8 short labels

### Color
- Black + cool gray as primary
- One clear academic accent color: Nature blue / IEEE blue / deep teal / dark red
- Accent color used only in one or two key locations

## Forbidden
- No cartoon, heavy texture, fake 3D, glossy SaaS illustrations
- No decorative stickers, giant empty hero-area whitespace
- No color scheme exceeding primary + 1 accent

## English Prompt Snippet
```
Nature / IEEE-inspired minimal academic presentation diagram, publication-quality scientific figure aesthetic, clean white or light gray background, precise method diagram with fine vector lines, clear restrained scientific color palette (black, gray, one accent: nature blue or teal), compact readable sans-serif labels, credible paper submission visual style, no decoration, no 3D, no cartoon
```
