# sketchnote — Warm Research Notebook Style / 温暖科研笔记风

Like a carefully organized research handout: bright, warm, refreshing, yet still professional. / 像一张被精心整理过的研究手抄报：明亮、温暖、清爽，但仍然专业。

## Atmosphere / 氛围
- Warm, personal, thoughtful / 温暖、个人化、有思考的温度
- Fresh and bright, like lecture notes / research handouts, not vintage old paper / 清爽明亮，像课堂讲义/研究手抄报，不是复古旧纸
- Not cold machine-generated graphics / 不像机器生成的冷冰冰配图
- Invites close inspection, like reading a carefully organized page of method notes / 让人想凑近看细节，像在读一页认真整理过的方法笔记

## Visual Specification / 视觉规范

### Base Color / 底色
- Bright light cream or buttercream (ref #FFF8EA / #FAF4E6 / #FFFDF7) / 明亮浅米白或奶油白（参考 #FFF8EA / #FAF4E6 / #FFFDF7）
- Clean paper surface, can have very subtle paper feel but no obvious texture / 纸面干净，可以有极轻微纸感，但不能有明显纹理
- No kraft paper, old parchment, vignettes, stains, sepia edges / 不要牛皮纸、旧羊皮纸、暗角、污渍、黄褐色边缘
- Overall impression: "warm and clean", not "old and vintage" / 整体观感："温暖干净"，不是"老旧复古"

### Lines / 线条
- Black hand-drawn style lines with natural ink stroke weight variation / 黑色手绘风格线条，有自然的笔触粗细变化
- Not perfect vector straight lines — slight irregularity allowed, but not sloppy / 不是完美的矢量直线——允许轻微不规则，但不能潦草
- Thick lines = core structure, thin lines = auxiliary connections / 粗线 = 核心结构，细线 = 辅助连接

### Text / 文字
- Handwritten style (English: sketch/casual fonts, Chinese: clear but slightly handwritten feel) / 手写体风格（英文选 sketch/casual 字体，中文清晰但略带手写感）
- Labels must be neat, not too messy to affect readability / 标注文字整洁，不能太潦草影响可读性
- Title font slightly larger and bolder than labels / 标题字体比标注字稍大、稍粗

### Emphasis / 强调
- Accent colors: deep blue #2C5F8A / coral red #D85C4A / olive green #5C6B4F / soft yellow #F3D36B / 强调色：深蓝 #2C5F8A / 珊瑚红 #D85C4A / 橄榄绿 #5C6B4F / 柔和黄 #F3D36B
- Emphasis methods: hand-drawn dashed borders, light color fills, marker underlines, local zoom circles / 强调方法：手绘虚线边框、淡彩填充、马克笔下划线、局部放大圈
- No large heavy color blocks — keep it light and clear / 不要大面积重色块，保持轻快清晰

### Information Density / 信息密度
- Main mechanism diagram should occupy 75%-85% of the canvas / 主体机制图应占画面 75%-85%
- When whitespace is excessive, prioritize: local zoom boxes, step mini-examples, dimension annotations, before/after comparisons / 留白过多时优先补充：局部放大框、步骤小例子、维度说明、before/after 对比
- One figure covers one main mechanism, but covers it thoroughly / 一张图只讲一个主机制，但要讲透

### Usable Comprehension Anchors (restrained) / 可用的理解锚点（克制）
- Small arrows (hand-drawn triangular arrows) / 小箭头（手绘三角箭头）
- Small circle/dot annotations / 小圈/小点标注
- Simplified small icons: magnifying glass, sticky note, lightbulb, bracket annotation / 简化小图标：放大镜、便签、灯泡、括号圈注
- Handwritten page number at bottom right / 右下角手写页码

## Forbidden / 禁止
- No real paper texture photos (too messy) / 不要真实纸张纹理照片（太乱）
- No narrative comics or character dialogue boxes / 不要变成剧情漫画、人物对话框
- No old diary, parchment, treasure map, or yellowed archives / 不要旧日记、羊皮纸、藏宝图或泛黄档案
- No childish cartoon style / 不要幼稚卡通风格
