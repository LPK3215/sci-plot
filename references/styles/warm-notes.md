# warm-notes — Bright Handwritten Notes Style / 明亮手记风

Goal: approachable, handcrafted feel, but not childish. Like carefully organized research study notes, suitable for science communication, teaching, and paper explanations. / 目标：亲和、有手作感，但不是幼稚。像一份被认真整理的科研学习笔记，适合科普、教学、论文讲解。

## Atmosphere / 氛围
- Warm, approachable teaching feel / 温暖、有亲和力的教学感
- Bright and clean, not vintage or distressed / 明亮干净，不是做旧复古
- Makes people want to keep reading / 让人愿意读下去

## Visual Specification / 视觉规范

### Canvas / 画布
- Warm cream white paper base (#FFF8EE / #FFFDF7) / 温暖奶油白纸底（#FFF8EE / #FFFDF7）
- Clean and bright, can have subtle paper feel / 干净明亮，可以有轻微纸感
- No dark tones, no distressed textures / 不要暗沉、不要做旧纹理

### Lines / 线条
- Hand-drawn style lines with stroke variation / 手绘风格线条，有笔触变化
- Slightly more orderly than sketchnote, but retains hand-drawn warmth / 比 sketchnote 更工整一点，但仍保留手绘温度

### Text / 文字
- Clear readable handwritten-style labels / 清晰可读的手写风格标注
- Title slightly larger and bolder / 标题稍大、稍粗
- Explanatory text short and friendly / 解释文字简短友好

### Color / 色彩
- Ink black + soft blue / coral / olive / muted yellow / 墨黑 + 软蓝/珊瑚/橄榄/柔和黄
- Light color highlights, not glaring / 淡彩高亮，不刺眼
- Accent colors soft but distinct / 强调色柔和但有区分度

### Usable Elements / 可用元素
- Hand-drawn arrows and connecting lines / 手绘箭头和连接线
- Small symbols: asterisks, lightbulbs, circle annotations / 小符号：星号、灯泡、圈注
- Local zoom boxes / 局部放大框

## Forbidden / 禁止
- No old parchment texture / 不要旧羊皮纸质感
- No vignettes, stains, yellowing / 不要暗角、污渍、泛黄
- No childish cartoon / 不要幼稚卡通
- No overly casual doodle style / 不要过于随意的涂鸦风

## English Prompt Snippet / 英文提示词片段
```
Warm research notebook educational diagram, clean cream paper background, hand-drawn but neat lines with mild ink variation, soft and friendly highlighter colors (muted blue, coral, olive, yellow), clear readable handwritten-style labels, approachable teaching/learning aesthetic, polished study notes style, no old parchment, no dark vignette, no childish cartoon, no messy doodles
```
