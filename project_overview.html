Core Workflow

六轮强制工作流

每一步不可跳过，确保提示词质量始终如一

0
⛔ 强制

确定分析目标

自动发现工作区中的 .tex / .md / .pdf / .py 文件，优先寻找 LaTeX 论文。支持本地路径、arxiv 链接、粘贴内容。

1
⛔ 强制

分析输入内容

先读取 analysis-guide.md，按论文/代码/算法/文档四类分析清单执行完整分析。覆盖核心创新、方法流程、关键实验、架构关系。

2
⛔ 强制

提取可图解内容

筛选适合画成图的内容：必须图解方法流程和核心机制，可选图解对比和架构，不图解相关工作和抽象概念。

3
⛔ 强制

询问用户

先读取 layout-rules.md，按页面角色组织方案。确认语言（中文/English）、视觉风格、生成张数、图片比例（16:9/4:3/2:3）、用途。

4
⛔ 强制

生成提示词

先读取 prompt-rules.md 和对应风格文件。严格使用模板格式，遵守构图三层次、颜色≤3种、每页必含/禁止清单等约束。

5
⛔ 强制

输出与汇总

生成 analysis.md、figure-plan.md、prompts/ 目录（每页一个 .md）、README.md 使用说明。所有产出统一放在 sci-plot-output/ 目录。

Features

核心能力

不只是提示词引擎，更是科研可视化的全流程解决方案

📄

多源输入

支持 LaTeX 论文（.tex）、Markdown 文档、PDF、Python 源码、arxiv 链接/ID、直接粘贴文本，自动发现工作区内容。

🎨

五种风格

论文框架图、温暖笔记、极简学术、明亮手记、商业简报，每种风格都有完整的颜色/构图/标注规范。

🔒

强制约束

颜色≤3种、禁止花哨装饰、禁止无方向箭头、禁止模糊描述。构图三层次（主结构→子模块→标注）确保信息清晰。

🌐

跨平台通用

提示词兼容 ChatGPT/DALL-E、豆包、Qwen、Midjourney、Stable Diffusion 等全部主流生图工具，不绑定任何特定平台。

📐

灵活比例

支持 16:9 横版、4:3 横版、2:3 竖版三种比例。默认竖版 2:3 用于方法图解，适配论文排版需求。

📋

结构化输出

每页独立 .md 文件，包含中文说明、构图描述、内容元素、English Prompt（核心，直接复制），以及平台使用指引。
